# Progetto di Controllo di un Sistema Dinamico (MATLAB)

Descrizione del progetto
Questo progetto riguarda la modellazione, analisi e progettazione di un sistema di controllo per un sistema dinamico reale scelto in accordo con il docente.

L’obiettivo è progettare un controllore in grado di garantire la stabilità asintotica del sistema in retroazione e soddisfare specifiche di prestazioni sia nel transitorio che a regime permanente.

---

Modellazione del sistema
- Scelta di un sistema dinamico reale
- Eventuale linearizzazione attorno a un punto di equilibrio
- Individuazione della variabile di controllo e della variabile controllata
- Ricavata la funzione di trasferimento del sistema

---

Obiettivi di controllo
- Stabilità asintotica del sistema in ciclo chiuso
- Prestazioni dinamiche (tempo di assestamento, sovraelongazione)
- Specifiche a regime permanente
- Analisi della robustezza del sistema

---

Progettazione del controllore
- Progettazione del regolatore in retroazione
- Eventuale azione di feedforward
- Analisi del ritardo massimo ammissibile
- Studio delle funzioni di sensitività e sensitività complementare

---

Simulazioni
Il sistema è stato simulato in MATLAB considerando:
- Configurazione in retroazione chiusa
- Presenza di saturazione all’ingresso
- Eventuali disturbi sul sistema

---

Risultati ottenuti
- Analisi della risposta nel dominio del tempo
- Analisi in frequenza (diagrammi di Bode)
- Verifica della stabilità del sistema
- Valutazione delle prestazioni con vincoli di saturazione

---

Strumenti utilizzati
- MATLAB
- Control System Toolbox
- (eventuale) Simulink
